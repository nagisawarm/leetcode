# 默认自动对普通字符串开启 intern机制，但是如果字符串中包含 空格 等特殊字符，就不会开启。
# 减少字符串的重复存储，从而节省内存空间。
# 在 Python 中，每当你创建一个新的字符串时，解释器会检查该字符串是否已经存在于字符串池（string pool）中。
# 如果是的话，它会返回已经存在的字符串的引用，而不是创建一个新的字符串对象。这个过程被称为 "字符串驻留"。
